Features

  • Add, list, and remove tasks from the command line
  • Mark tasks as completed
  • Persists tasks using JSON storage
  • Simple and lightweight, no extra dependencies

Installation

To install and use TaskBuddy, follow these steps:

1. Download the binary for your platform
2. Make it executable: chmod +x taskbuddy-aarch64
3. Run the program: ./taskbuddy-aarch64
